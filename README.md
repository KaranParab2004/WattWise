# Solar Energy Calculator

A web-based solar panel calculator and selection tool for final year project.

## Features

- Installation type selection (Residential, Industrial, Commercial, Ground Mounted)
- Solar power requirement calculator based on electricity bill
- Energy provider selection
- Solar panel manufacturer comparison
- Inverter selection
- Cost breakdown with subsidy calculation
- ROI and break-even analysis

## How to Run

1. Open `index.html` in a web browser
2. No build process or dependencies required

## Project Structure

```
├── index.html      # Main HTML structure with all screens
├── styles.css      # Complete styling
├── script.js       # Interactive functionality
└── README.md       # Documentation
```

## Screens

1. Installation Type Selection
2. Calculator Form (pincode, provider, billing)
3. Power Requirement Results
4. Panel & Inverter Selection
5. Technical Comparison
6. Final Selection with Cost Breakdown

## Technologies

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- Vanilla JavaScript (ES6+)

## Customization

- Update provider logos in the provider grid
- Modify calculation formula in `script.js`
- Add more brands/inverters in screen 4
- Customize colors in `styles.css`
